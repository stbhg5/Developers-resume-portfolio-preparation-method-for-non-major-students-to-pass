# 추상어의 저주 : 이력서를 추상적으로 적지 말고 구체적으로 적어라

1. 숫자를 활용해서 표현?
추상적)
• 테스트 코드 작성
• 인덱스를 활용한 쿼리 성능 최적화

구체적)
• E2E 테스트 코드 52개, 단위 테스트 코드 120개 작성
• 인덱스를 활용해 쿼리 성능을 1000ms에서 50ms로 20배 향상


2. 구체화 양식 (프로그래밍 실력 평가 2단계를 자연스럽게 녹여냄)
그림을 통한 과정 설명(도식화) → 도입 배경(원인/문제점) → 사용 이유(특징/장단점/작동원리) → 성과(숫자/결과물)
