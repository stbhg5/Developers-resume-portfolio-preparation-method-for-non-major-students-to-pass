[프로그래밍 실력을 평가하는 4단계]


1. 프로그래밍 실력을 평가하는 4단계
Level 1. 단순 기능 구현
ex) 결재 기능 구현, Restful API 구성, 관리자 페이지 구축 및 운영, AWS RDS의 MySQL을 데이터베이스로 사용, DB 구축 및 설계
-> 이력서에 1단계 내용만 작성하는 경우가 많음

Level 2. 그 기술을 왜 사용했는지, 기술의 특징/장단점은 어떻게 되는지, 기술의 작동 원리

Level 3. 유지보수하기 좋게 코드 작성 (객체지향, 클린코드, 테스트 코드 등)

Level 4. 성능 트러블 슈팅 및 개선, 대용량 트래픽 처리 역량

서류 전형에서 합격하려면, 1단계보다 2, 3, 4단계 내용을 많이 작성해야 한다.
공부나 프로젝트할 때 2, 3, 4단계 내용을 경험할 수 있게 학습해야 내용을 작성할 수 있다.
